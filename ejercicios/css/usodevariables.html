Uso de variables

El uso de variables es algo que puede resultar de lo más útil. La idea es crear una variable con el selector :root para, si en un futuro es necesario, hacer los cambios sólo en ese selector.

Las variables se introducen dentro de las llaves ({}) e inician con dos guiones, por ejemplo "--colorFondo" (sin las comillas), seguido de dos puntos y la variable, siendo un color en el caso de este ejemplo. Si en el cuerpo le asignamos una variable a una etiqueta en vez de un valor, los cambios futuros serán más sencillos. El código de esta página de ejemplo de uso de variables quedaría así:

:root {
    --colorFondo: rgb(211, 232, 241); --colorPrimario: darkcyan; --otro: orange;
}

body {
    background-color: var(--colorFondo, salmon); color: var(--colorPrimario);
    margin: 30px;
}

p {
     color: var(--otro);
}

A partir de ahora, si queremos cambiar el color de fondo o el primario, bastará con cambiar el valor de --colorFondo o --colorPrimario dentro de :root.
