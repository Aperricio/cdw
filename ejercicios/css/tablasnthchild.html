Tablas con colores diferentes de dos en dos y sin separación

Hay algo que queda muy estético y ayuda a la lectura de las tablas: que filas o columnas sean de colores diferentes. La primera del color 1, la segunda del color 2, la tercera del color 1, la cuarta del color 2... Eso se consigue con nth-child. "odd" son las impares, e "even" son las pares.

Las tablas ya sabemos cómo se hacen. En el CSS de lo anterior, para elegir que las filas sean de diferente color tenemos que usar el selector de etiqueta "tr" seguido de dos puntos (:) y "nth-child()". Entre paréntesis van los "odd" o "even" anteriormente mencionados. El CSS quedaría así:

table {
      text-align: center;
}

th {
      background-color: purple;
      color: white;
}

tr:nth-child(odd) {
      background-color: rgb(108, 179, 207);
}
tr:nth-child(even) {
      background-color: skyblue;
}

De lo anterior, a la table en general se le ha centrado el texto, a las casillas de cabecera se le ha puesto color de fondo violeta y blanco para el texto y luego hemos definido dos reglas para las filas (tr). En las impares (odd) le hemos añadido un azul y en las pares (even) otro, para que se note la diferencia pero no demasiado.

Tabla alternando el color de las columnas

Lo único que hay que cambiar aquí son los "tr" (filas) por "td", que son datos y hay que recordar que las columnas están formadas por datos (td).

El CSS sería el siguiente

table ~ table td:nth-child(odd) {
      background-color: rgb(108, 179, 207);
}

table ~ table td:nth-child(even) {
      background-color: skyblue;
}

caption {
      background-color: grey;
      padding: 10px;
}

He usado el símbolo "~" para elegir "la siguiente tabla tras la primera tabla" para poder dejarlo todo en la misma página, o de lo contrario me dejaría las dos tablas iguales. El "caption", pues eso, indica que me lo he inventado todo por falta de ideas.

El color RGB se ha elegido en base al "skyblue", un color cercano al que podemos acceder desde Visual Studio Code.

En cuanto a cómo eliminar los bordes de las casillas, hay que usar en "table" la declaración (propiedad: valor;) border-collapse: collapse. Con eso conseguimos juntar las casillas. Los bordes se añaden después con las declaraciones border-LADO: 2px solid purple a donde queramos añadir un borde, ya sea de la tabla o de una casilla.

Y así, queridos amigos, se consigue este efecto en las tablas.
