Función con parámetro

        <script>
            function minuto(tope = 0) {
                let fecha = new Date();
                let elMinuto = fecha.getMinutes();
                if ( elMinuto < tope ) {
                    alert(elMinuto + " es menos que " + tope)
                }
                else {
                    alert(elMinuto + " es más que " + tope)
                }
            }
            minuto();
            minuto(30);
            minuto(59);
        </script>

Al ponerle un valor por defecto, en la primera pasada mira ese valor aunque la función esté vacía.
