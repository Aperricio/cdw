Condicional con negación con if

Sólo verás el "alert" si estás en la segunda parte de un minuto. Si no, irás a mi página principal

        <script>
            let fecha = new Date();
            if (! (fecha.getSeconds() < 30)) {
                alert("SÍ estás en el segundo medio minuto")
            }
            else {
                alert("NO estás en el segundo medio minuto, así que te piras");
                (window.location.href="http://pablo.tomateun.com");
            }
        </script>
