Uso de <kbd> y <code>

Este ejercicio o anotación me lo he hecho para mi mismo, no se dio en el curso ni sirve de mucho, pero sí es interesante. Cuando hablamos de las primeras etiquetas, hablamos de la carga semántica, es decir, lo que le dice al navegador qué es qué, y eso también lo usa Google para analizar el contenido HTML y valorar si un documento es mejor o peor.

Por ese motivo hay que usar etiquetas cuando lo requieran. ¿Para qué podemos usar <kbd>? La etiqueta significa teclado (keyboard), y se usa para indicar qué teclas tenemos que presionar para conseguir algo. Por ejemplo, en Linux para sacar un terminal tenemos que presionar Ctrl + Alt + T. Además, si le damos forma, parecen teclas. Lo siguiente, lo que explica cómo hacerlo, se escribiría usando <code>:

<kbd style="background-color: #555; color: white; border: 3px double grey; border-radius: 5px;">Ctrl</kbd> + <kbd style="background-color: #555; color: white; border: 3px double grey; border-radius: 5px;">Alt</kbd> + <kbd style="background-color: #555; color: white; border: 3px double grey; border-radius: 5px;"> T </kbd>

También se puede crear un estilo CSS que quedaría así:

kbd {
    background-color: #555;
    color: white;
    border: 3px double grey;
    border-radius: 5px;
}

No es algo que nos vaya a salvar la vida, pero es semánticamente correcto y queda mejor si se tiene que compartir contenido a modo de tutorial.
