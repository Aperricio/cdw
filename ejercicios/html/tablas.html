Tablas

Las etiquetas para las tablas son:

  • table: toda la tabla.
  • th: datos de cabecera. Sale en negrita. También van dentro de tr.
  • tr: fila.
  • td: datos.

Las columnas las forman el número de filas (tr). Esto sería en una tabla "regular". En las irregulares, el atributo "colspan" indica el ancho (en número de celdas) de la celda y "rowspan" indicaría lo alto (también en número de celdas).

Estructura en las tablas largas

Cuando la tabla es larga, se suele hacer con una estructura:

  • table: etiqueta para toda la tabla.
  • caption: le pone un título a la tabla.
  • thead: aquí irían las partes importantes o titulares, lo que queda más claro cuando usamos más de uno.
  • tbody: aquí irían los datos generales.
  • tfoot: aquí irían datos como el resultado de una suma.

Puede ser importante añadir la estructura para facilitar su decoración por medio del CSS. Dentro de thead, tbody y tfoot hay que meter la información (td).
