Ejercicio de fotos



La etiqueta <img> sirve para añadir fotos. src sería la fuente (SouRCe), y el atributo alt se usa para que aparezca un texto si la imagen no pudiera cargarse; también lo usan programas de accesibilidad. Si el valor es algo como https://via.placeholder.com/640x480.png, podemos poner un tamaño para que lo reserve.
