Pruebas de enlaces

  • Patear es mejor que googlear.
  • ftp-simple, extensión recomendada para gestionar el FTP desde Visual Studio Code. Si se prefiere gestionar desde una copia de seguridad (carpeta) de nuestro equipo, mejor usar SFTP de liximomo. Otra extensión recomendada es Code Spell Checker, lo que es un corrector. Para el español hay que instalar también su paquete. Mención especial al Live Preview de Microsoft que nos permite ver en tiempo real los cambios realizados en un documento HTML.
  • Sistemas operativos:
    • Manjaro: basado en Arch Linux, pero con tres ramas a modo de cortafuegos (Unstable, Testing y Stable). Software siempre actualizado, como muy tarde, tras dos semanas de su lanzamiento.
    • Ubuntu: una de las distribuciones más usadas. En su edición GNOME es más sencillo y estable, pero más pesado.
    • Kubuntu: sabor oficial de Ubuntu, pero con software de KDE. Se puede ver algún bug, pero el software de KDE es más ligero y sus aplicaciones más productivas.
  • vokoscreenNG: para grabar la pantalla, el micro, el sonido del equipo e incluso la webcam. Es multiplataforma y de código abierto.
